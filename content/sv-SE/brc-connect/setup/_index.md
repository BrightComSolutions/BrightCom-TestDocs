# BRC Connect Setup

This guide walks you through the installation and initial setup of BRC Connect.

## Prerequisites

Before installing BRC Connect, ensure you have:

- Microsoft Dynamics 365 Business Central (supported versions)
- Appropriate permissions to install apps
- Valid BRC Connect license

## Installation Steps

### 1. Install from AppSource

1. Go to [Microsoft AppSource](https://appsource.microsoft.com/)
2. Search for "BRC Connect"
3. Click **Get it now**
4. Follow the installation wizard

### 2. Initial Configuration

After installation, complete the initial setup:

1. Open **BRC Connect Setup** from the Business Central search
2. Configure your connection settings
3. Test the connection
4. Save your configuration
