## Welcome to BRC Core

BRC Core enhances your Business Central experience with automated background management, currency updates, and administrative tools. This guide helps first-time users understand and access BRC Core features.

## First Time Access

### Checking Your Permissions

Before using BRC Core, verify you have appropriate access:

1. **Search for BRC features** (Alt+Q, type "BRC")
2. **If you see BRC Core pages**, your permissions are configured
3. **If no results appear**, contact your administrator for permission assignment

### Understanding Feature Availability

BRC Core uses conditional feature management. Some features may not be available based on:
- Your user role
- Company settings  
- Custom business conditions

## Basic Navigation

### Accessing BRC Core Features

**Method 1: Search (Recommended)**
1. Press **Alt+Q** to open search
2. Type **"BRC"** to see all available features
3. Select the desired feature

**Method 2: Menu Navigation**
Some features appear in standard Business Central menus based on your role center.

### Common Starting Points

**For General Users:**
- **BRC Core Feature Management** - See available features
- **Background Monitor pages** - Check job status (if applicable)

**For Administrators:**
- **BRC Core BM Setup** - Background monitoring configuration
- **BRC Admin Toolbox** - System administration tools

## Understanding the Interface

### Feature Management Page

The Feature Management page shows:
- **Available Features**: What BRC Core features exist
- **Feature Status**: Which features are enabled for you
- **Conditions**: Why features are available or restricted

**Interpreting Feature Status:**
- ✅ **Available**: You can use this feature
- ❌ **Restricted**: Feature not available based on conditions
- ⚠️ **Conditional**: Availability depends on context

### Background Monitor Interface

If you have access to job queue monitoring:
- **Green indicators**: Jobs running normally
- **Yellow indicators**: Jobs with warnings
- **Red indicators**: Jobs with errors requiring attention

### Currency Rate Pages

Currency management shows:
- **Current Rates**: Latest exchange rates
- **Update Status**: When rates were last refreshed
- **Service Status**: External service connectivity

## Basic Operations

### Viewing System Status

**Check Background Job Status:**
1. Search for "**BRC Core BM Action Entries**"
2. Review recent monitoring activity
3. Look for error notifications or warnings

**Check Currency Rate Status:**
1. Search for "**BRC Curr Exch Rate**" 
2. Review rate update history
3. Verify rates are current

### Using Feature Management

**View Your Available Features:**
1. Open "**BRC Core Feature Management**"
2. Review the list of available features
3. Note any restricted features and their conditions

**Check Feature Conditions:**
1. Select a feature in Feature Management
2. Choose "**Feature Conditions**" to see why it's available/restricted
3. Contact administrator if access is needed

### Basic Admin Tools (If Available)

**System Diagnostics:**
1. Open "**BRC Admin Toolbox**"
2. Use diagnostic tools to check system health
3. Review record counts and system statistics

## Understanding BRC Core Workflows

### Automated Processes

BRC Core runs several automated background processes:

**Currency Rate Updates:**
- Runs on schedule (typically daily)
- Updates exchange rates from external sources
- Notifications sent if updates fail

**Background Monitoring:**
- Continuously monitors job queue entries
- Alerts on job failures or long-running processes
- Provides automated retry capabilities

**Price Book Updates:**
- Calculates prices based on configured rules
- Runs at scheduled intervals
- Updates product pricing automatically

### User Interactions

**Monitoring Dashboards:**
- Review system status through monitoring pages
- Check for alerts or notifications
- Verify automated processes are running correctly

**Administrative Tasks:**
- Configure features (if administrator)
- Set up conditions for feature availability
- Manage user access to specific functionality
