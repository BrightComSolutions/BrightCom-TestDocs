## Overview

This user guide provides step-by-step instructions for using BRC Core features in your daily Business Central operations. The guide covers common tasks, workflows, and best practices for each major feature area.

## Getting Started

Before using BRC Core features, ensure:
- Your user account has the appropriate permission sets assigned
- Initial configuration has been completed by your administrator
- You understand which features are enabled for your role

## Core Feature Areas

BRC Core provides several integrated feature areas that work together to enhance your Business Central experience:

### Feature Management
Controls which features are available based on user, company, or custom criteria.

### Background Monitor
Monitors and manages job queue entries with automated error handling and notifications.

### Currency Exchange Rate Management
Automates currency rate updates from multiple external sources.

### Price Book Management
Manages automated price calculations and updates for your product catalog.

### GDPR Management
Handles customer data anonymization and deletion for compliance requirements.

### Admin Toolbox
Provides administrative utilities for system diagnostics and management.

## User Roles and Permissions

Different user roles have access to different BRC Core features:

### End Users
- View monitoring status
- Access feature-controlled functionality
- Basic price and currency information

### Administrators
- Full feature configuration
- Background monitor management
- User and permission management
- System diagnostics and tools

### Compliance Officers
- GDPR management tools
- Data anonymization controls
- Compliance reporting

## Navigation and Access

### Finding BRC Core Features

All BRC Core features are accessible through Business Central's standard search:

1. **Press Alt+Q** to open search
2. **Type "BRC"** to see all available BRC Core features
3. **Select** the desired feature from the search results

### Common Search Terms

- "BRC Core Feature Management" - Feature control
- "BRC Core BM Setup" - Background monitor configuration  
- "BRC Curr Exch Rate" - Currency rate management
- "BRC Admin Toolbox" - Administrative tools
- "BRC Core GDPR" - Data privacy tools

### Role Center Integration

Some BRC Core features integrate with Business Central Role Centers, appearing as:
- Activities tiles for monitoring
- Reports and analytics
- Quick access links

## Best Practices for Users

### Daily Operations
- Check Background Monitor status regularly if managing job queues
- Verify currency rates are updating as expected
- Monitor feature availability notifications

### Data Management
- Understand GDPR implications before customer data changes
- Follow established procedures for price updates
- Report unusual system behavior to administrators

### Security
- Only use features within your assigned role
- Report permission issues to administrators
- Follow company policies for data access and modification
