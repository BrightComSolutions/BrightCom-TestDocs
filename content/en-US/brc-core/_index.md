## Overview

BRC Core is a foundational Microsoft Dynamics 365 Business Central app that provides essential background management capabilities and core extensions to enhance your ERP operations. The app focuses on automating routine tasks, monitoring system health, and providing administrative tools that improve system reliability and efficiency.

## Key Features

- **Background Monitor**: Automated monitoring and management of job queue entries with error handling and notification capabilities
- **Currency Exchange Rate Management**: Automated currency rate updates from multiple external sources including ECB, Riksbank, Nationalbanken, and others
- **Feature Management**: Conditional feature enablement based on user, company, or custom criteria for controlled rollouts
- **PriceBook Engine**: Automated price calculation and update system for maintaining current pricing
- **GDPR Management**: Customer data anonymization and deletion tools for compliance requirements  
- **Admin Toolbox**: Comprehensive administrative utilities for system diagnostics and management
- **Business Central Extensions**: Enhanced functionality for inventory valuation, item dimensions, VAT posting, customs documents, and more

## Business Value

BRC Core addresses several critical business needs:

- **Reduced Manual Work**: Automates currency updates, price calculations, and background task monitoring
- **Improved System Reliability**: Proactive monitoring and error handling for background processes
- **Enhanced Compliance**: GDPR data management tools for regulatory requirements
- **Administrative Efficiency**: Centralized toolbox for system administration and diagnostics
- **Flexible Feature Control**: Granular control over feature availability based on business conditions
